# Sound Assets

## Order Alert Sounds

Place your custom alert sound files here:

- `new_order.mp3` - Played when a new order arrives
- `order_ready.mp3` - (Optional) Played when order is ready for pickup
- `order_cancelled.mp3` - (Optional) Played when order is cancelled

## Recommended Specifications

- **Format**: MP3 or WAV (MP3 recommended for smaller size)
- **Duration**: 1-3 seconds
- **Volume**: Normalized to -3dB to -6dB
- **Sample Rate**: 44.1kHz
- **Bit Rate**: 128-192 kbps

## Free Sound Sources

You can get free notification sounds from:
- https://notificationsounds.com/
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/

## Example Sounds

If no custom sounds are provided, the app will use a built-in system beep sound.

## Testing

Test your sounds in the merchant app:
1. Enable sound alerts in Orders page
2. Create a test order from the customer app
3. Verify the sound plays in the merchant console
